# Mon Portfolio Personnel 🚀

## 📝 Description
Ceci est mon portfolio professionnel présentant mes projets et mes compétences en développement web. 
L'objectif est d'offrir une interface claire pour me découvrir et me contacter.

**Fonctionnalités :**
- Design moderne et responsive.
- Section "about" "Projets" "contact"

## 🛠 Technologies utilisées
- **HTML5** & **CSS3**
- **Tailwind CSS** (Framework CSS)
- **JavaScript** (Animations et DOM)
- **GitHub Pages** (Hébergement)
